# Анализ глобального развития: Всемирный банк и HDI

Учебный проект по визуализации данных. Разведочный анализ двух открытых наборов данных — социально-экономических индикаторов Всемирного банка и Индекса человеческого развития ООН — с акцентом на различия между регионами и динамику развития с 1990 по 2023 год.

> **Стек:** Python · pandas · numpy · plotly · geopandas · matplotlib

## Данные

| Источник | Файл | Содержимое |
|---|---|---|
| World Bank | `JobsCSV.csv` | 168 индикаторов по 242 странам (ВВП на душу, грамотность, доступ к электричеству, урбанизация, налоги и др.) |
| UNDP HDR | `hdr-data.xlsx` | HDI для 193 стран, 1990–2023 |

Оба файла лежат в этом же репозитории и подгружаются в ноутбуке напрямую.

## Что в ноутбуке

1. **Преобразование данных Всемирного банка** — функция `get_df_from_year` разворачивает long-формат в wide и возвращает таблицу «страна × индикатор» за нужный год.
2. **Доступ к электричеству в 1990** — choropleth-карта мира + pie-chart с глобальной долей населения с/без доступа.
3. **Корреляции (2010)** — heatmap для четырёх показателей: грамотность, налоги, ВВП на душу, урбанизация.
4. **Динамика HDI для БРИКС и постсоветских стран** — line plot с 1990 по 2023.
5. **Мировая карта HDI с анимацией по годам** — интерактивная choropleth.
6. **Кластеризация стран по профилю развития** - сравнение работы kmeans, dbscan, meanshift через PCA, применение silhouette score и метода локтя.
7. **Выводы** — ключевые наблюдения, ограничения, возможные направления развития.

## Ключевые наблюдения

- В 1990 году только **71,4%** мирового населения имело доступ к электричеству. К 2023 году средний мировой HDI вырос с ~0.6 до **0.74**.
- Самый драматический рост HDI среди анализируемых стран — у **Китая** (0.491 → 0.797).
- **Урбанизация и ВВП на душу населения** коррелируют сильнее всего (0.65); налоговые поступления почти не коррелируют с другими показателями развития.
- Геополитические события (расширение ЕС, кризисы, конфликты) оставляют видимые следы на траекториях HDI отдельных стран.
